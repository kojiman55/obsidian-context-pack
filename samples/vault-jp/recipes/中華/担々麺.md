# 担々麺

![担々麺](https://upload.wikimedia.org/wikipedia/commons/thumb/4/4a/Soupless_dandan_noodles_%28mild%29_of_175do_DENO_Tantanmen.jpg/960px-Soupless_dandan_noodles_%28mild%29_of_175do_DENO_Tantanmen.jpg)

ゴマの濃厚なスープと辛みが絡む本格担々麺。しびれる辛さがクセになる。

## 材料（4人分）

| 材料 | 分量 |
|---|---|
| 中華麺 | 2玉 |
| 豚ひき肉 | 150g |
| チンゲン菜 | 2株 |
| 長ねぎ | 1/2本 |

### 調味料

| 調味料 | 分量 |
|---|---|
| 練りごま | 大さじ4 |
| 豆板醤 | 大さじ1 |
| 醤油 | 大さじ2 |
| 酢 | 大さじ1 |
| 砂糖 | 小さじ1 |
| 鶏ガラスープ | 400ml |
| ラー油 | 適量 |
| 花椒 | 適量 |
| すりごま | 大さじ2 |

## 手順

1. ひき肉を炒めて甜麺醤・醤油で味付けし、肉みそを作る。
2. 練りごま・醤油・酢・砂糖・豆板醤をよく混ぜてタレを作る。
3. スープを温め、タレを加えてよく混ぜる。
4. 麺を茹で、スープを注ぎ、肉みそ・チンゲン菜・長ねぎをのせる。
5. ラー油と花椒を仕上げにかける。

## コツ・ポイント

- 練りごまは水で少し伸ばしておくと混ざりやすい。
- スープは沸騰させずに温める。

## メモ

辛さは豆板醤とラー油で調整。汁なし担々麺にするとより濃厚な味わいになる。
