# 酢豚

![酢豚](https://upload.wikimedia.org/wikipedia/commons/thumb/2/2e/Sweet_and_sour_pork.jpg/960px-Sweet_and_sour_pork.jpg)

揚げた豚肉と野菜に甘酢あんをからめた、子どもから大人まで人気の中華料理。

## 材料（4人分）

| 材料 | 分量 |
|---|---|
| 豚バラ肉（または肩ロース） | 300g |
| ピーマン | 2個 |
| にんじん | 1/2本 |
| 玉ねぎ | 1/2個 |
| パイナップル（缶） | 4切れ |
| 片栗粉 | 適量 |

### 調味料

| 調味料 | 分量 |
|---|---|
| 酢 | 大さじ3 |
| 砂糖 | 大さじ3 |
| ケチャップ | 大さじ2 |
| 醤油 | 大さじ1 |
| 鶏ガラスープ | 100ml |
| 片栗粉（あん用） | 大さじ1 |

## 手順

1. 豚肉は一口大に切り、醤油・酒・生姜で下味をつけて片栗粉をまぶす。
2. 170℃の油で揚げ、一度取り出してから180℃で二度揚げする。
3. 野菜は乱切りにして素揚げする。
4. 酢・砂糖・ケチャップ・醤油・スープを合わせてあんを作る。
5. 揚げた豚肉・野菜・パイナップルにあんをからめる。

## コツ・ポイント

- 豚肉は二度揚げでカリッと仕上げる。
- あんは全体にからめる直前に作る。

## メモ

パイナップルを入れると酸味と甘さが増す。苦手な場合は省いてもよい。
