# ミネストローネ

トマトベースの野菜たっぷりイタリアンスープ。体にやさしい一品。

## 材料（4人分）

| 材料 | 分量 |
|---|---|
| 玉ねぎ | 1個 |
| にんじん | 1本 |
| セロリ | 1本 |
| ズッキーニ | 1本 |
| じゃがいも | 1個 |
| ベーコン | 50g |
| トマト缶 | 1缶 |
| 白いんげん豆（缶） | 1缶 |
| ショートパスタ | 50g |

### 調味料

| 調味料 | 分量 |
|---|---|
| 野菜ブロス | 600ml |
| オリーブオイル | 大さじ2 |
| にんにく | 1かけ |
| 塩こしょう | 適量 |
| パルミジャーノ | 適量 |

## 手順

1. 野菜はすべて1cm角に切る。
2. 鍋でオリーブオイルを熱し、にんにく・ベーコンを炒める。
3. 玉ねぎ・にんじん・セロリを加えて5分炒め、残りの野菜も加える。
4. トマト缶・ブロス・豆を加えて20分煮込む。
5. パスタを加えて袋の表示時間通り煮て、塩で味を整える。

## コツ・ポイント

- 野菜は同じ大きさに切ると均一に火が通る。
- 豆を入れると食べ応えと栄養価が増す。

## メモ

パルミジャーノを削って仕上げに。翌日は味がなじんでさらに美味しい。
