# 豚汁

豚肉と根菜たっぷりの具だくさん味噌汁。体が温まる冬の定番。

## 材料（4人分）

| 材料 | 分量 |
|---|---|
| 豚バラ薄切り肉 | 150g |
| 大根 | 5cm |
| にんじん | 1/2本 |
| ごぼう | 1/2本 |
| こんにゃく | 1/2枚 |
| 長ねぎ | 1本 |

### 調味料

| 調味料 | 分量 |
|---|---|
| だし | 800ml |
| 味噌 | 大さじ3〜4 |
| 酒 | 大さじ1 |
| ごま油 | 小さじ1 |

## 手順

1. 野菜はすべて食べやすい大きさに切る。ごぼうは水にさらす。
2. 鍋でごま油を熱し、豚肉を炒める。
3. 野菜を加えて炒め、全体に油が回ったらだしを加える。
4. アクを取りながら野菜が柔らかくなるまで15分煮る。
5. 火を弱めて味噌を溶き入れ、長ねぎを加えてひと煮立ちさせる。

## コツ・ポイント

- 味噌は沸騰させると香りが飛ぶので注意。
- ごぼうはしっかりアクを抜く。

## メモ

七味唐辛子を振るとアクセントになる。翌日も美味しい。
